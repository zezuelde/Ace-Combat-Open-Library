# Ace-Combat-Open-Library
## An Open Library of all preserved assets from the Ace Combat series (1995 - XXXX)

This was made to help all people of all kind, whether modders, livery makers, animators, and a whole bunch of others (with a disclaimer of maybe consulting the creators) to being able to use the assets for their own personal use.
  
> How do I contribute/add more assets that I've found/ripped?

You can either email the following: zezuely10@gmail.com and ask to be added to become a contributor OR the latter; email the stuff you've ripped/found and I can commit it into the main branch itself (you can ask for credit or no credit, your choice)  

> Where can I find more of these libraries?

  Either checking my profile and going into the **Repositories** tab or going into this repositorys wiki which is [here](https://github.com/zezuelde/Ace-Combat-Open-Library/wiki/Other-Libraries--And-Extras)
  
> Why are all the librarys seperate and some of the models contained in ZIP files?!

1. Just incase we meet the space cap of githubs upload limit.  
2. For compressing sizes.

> [!NOTE]
> Aside from the reasons already given, I've wanted to add more. The Ace Combat series and its old nature is seeming to cripple under its own age and I've started this to preserve the essence of it, with anything I can do. This also includes relatively anything in the games like Models, Soundtracks (if possible), Levels, cut/unused content (if any), Textures and also possibly the source code of such games. I've started to expand this project to all of the games which may be hard to obtain but worth a shot, I'm currently on a task to replace all the assets with my own that I've extracted or others that volunteered to help out.
